What We Built

Platform at a Glance

01

Peer-to-Peer Learning

SkillSwap connects individuals with complementary skills. A designer who wants to code meets a coder who wants to design — and both grow together through mutual exchange.

02

Gamified Assessments

The Take Test module offers Concept Match MCQs, Debugging Race challenges, Code Rearrangement puzzles, and Code Completion exercises with timed scoring and leaderboards.

03

Smart Matching System

Users list skills they offer and skills they want. The platform intelligently pairs learners with suitable peers based on complementary interests, enabling efficient and personalized connections.

04

Full-Stack Architecture

Built with React frontend, Node.js + Express backend, Supabase for authentication, and a structured database storing users, sessions, tests, messages, and reviews.
