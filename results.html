Chapter 4

Results & Discussion

SkillSwap successfully implements its core vision of peer-to-peer skill exchange with a complete, functional platform featuring real-time chat, gamified assessments, user profiles, and session management.

Implemented Features Overview

The SkillSwap platform has been developed and deployed as a functional full-stack web application. All primary features described in the objectives have been implemented and tested. Below is a summary of the key results and the working modules of the platform:

  • User registration, login, and authentication via Supabase (email/password and OAuth)
  • Profile management with skills offered, skills wanted, bio, links, and badge display
  • Real-time peer messaging with emoji, file attachment, and calendar scheduling
  • Gamified Take Test module with five challenge types and live leaderboard scoring
  • Test results page with score, star rating, rank position, and answer review
  • Peer review and feedback system after each completed learning session

Take Test Module — 5 Game Types

The gamified assessment module is one of SkillSwap's most distinctive features, offering five different challenge formats designed to validate learner skills across conceptual understanding and practical coding ability:

  • Concept Match (MCQ) — A concept description is shown; select the correct concept title. Fast responses earn bonus points. Timed at 5–10 minutes.
  • Debugging Race — Identify and fix bugs in short code snippets before time runs out. Speed + correctness = final score. Timed at 5–10 minutes.
  • Code Rearrangement — Drag and drop shuffled lines of code into the correct order to form a working program. Tests logical sequencing ability.
  • Code Completion — Fill in missing keywords and syntax by drag and drop. Complete incomplete code by dragging missing tokens into place. Accuracy and speed both rewarded.
  • Fill in the Blank Coding — Complete code snippets with missing syntax elements, testing syntax familiarity and precision under time pressure.

Scoring & Ranking System

The test module features a comprehensive performance measurement system that motivates continued engagement through transparent metrics and peer comparison:

  • Points Score — Awarded for each correct answer, with bonus points for fast responses
  • Star Rating (1–5) — Reflects overall performance quality in the session
  • Rank Position — Shows the user's standing among all peers who completed the same test (e.g., #64 out of all participants)
  • Answer Review — Full review of each question showing your answer, the correct answer, and an explanation of why

Messaging System Results

The real-time chat system was successfully implemented, supporting multi-user conversations between matched peers. The messages interface shows recent conversations with peer details (name, interest), timestamps, and full chat history. The conversation view includes a message input bar with emoji picker, file attachment, calendar scheduling, and a send button. Messages are styled to clearly distinguish sent (right-aligned, purple) from received (left-aligned, dark) messages — following modern chat UI conventions familiar to users.

Discussion

The SkillSwap platform successfully demonstrates the concept of a peer-to-peer learning environment with a complete, working implementation. The gamified test module proved particularly engaging during testing — the combination of timed challenges, immediate feedback, and peer ranking created a compelling loop that encourages repeated practice and skill improvement. The messaging system enables the natural peer interaction that is central to the platform's philosophy, allowing learners to schedule sessions, share resources, and follow up after completed exchanges.

The architecture's modular design — with clearly separated frontend, backend, business logic, and database layers — means individual components can be upgraded independently without disrupting the whole system, which is essential for future enhancements like AI-based matching and mobile application development.
